# Data-Analytics-Project
# Carbon Emissions Analysis & Visualization

## Project Overview
End-to-end data analytics project analyzing global CO₂ emissions 
from agrifood systems across 236 countries from 1990 to 2020.

## Key Steps
- Advanced missing data analysis (MCAR, MAR, MNAR classification)
- MICE imputation using Scikit-learn IterativeImputer with Linear Regression
- Exploratory Data Analysis — temporal trends, country contributions, sector drivers
- Feature engineering — 16 new calculated columns for Power BI
- Interactive Power BI dashboard with KPI cards, slicers, and 6 visuals

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI Desktop 

## Dataset
- 6,965 records × 31 original columns
- 236 countries, 31 years (1990–2020)
- Source: Agrofood CO2 Emission dataset from the Food and Agriculture Organization (FAO, USA) 

## Dashboard Features
- KPI cards — Total Emissions, Avg Temperature Rise, YoY Growth, Countries
- Dual-axis trend chart — Emissions vs Temperature
- Top 10 countries by cumulative emissions
- Sector breakdown by decade (stacked bar)
- Emission share by sector (donut chart)
- Top emission sources (horizontal bar)
- Interactive slicers — Region and Year range
